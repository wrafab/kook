Laser Cutting 360




This week's task is to work with laser cutting!


For practice, I designed a cardboard box using MakerCase.

I measured the length, width, and the thickness of the cardboard first. Then, I adjusted the box width, height, and depth as I desire. I chose finger edge joints.



Using Coreldraw, I added some letters to raster.



This is the box I created using laser cutting!
